# AI Usage Log

This file contains information about how AI has been used for assistance during this project. This documents the workflow of the project as I do not have a complete and detailed log of every use. The tools used are Copilot and GPT. I have also done searchs on Google, which sometimes has provided AI generated answeres.

The JavaScript Frameworks course started with reading the content in the course modules. After reading a module section, I did the exercise connected to that section. During this process with learning the content, AI has been used to explain different strategies and parts of code. It has also been used to create different examples than the ones provided in the course material to get a clearer overview of the different elements and to provide feedback to the exercise works.

After starting on the course assignment, I have had several visits back to the course material while using AI to explain different elements as before. It has also been used to explain different error messages that I have encountered on the way.
